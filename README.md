# shuttleBox
> can select three option and handle
> depend on bootstrap
## use
``` 
this.shuttleBox = new ShuttleBox({
                body:'#shuttleBox',
                async:false,
                provinceList:['北京','上海','浙江省'],
                cityList:['美食','酒店','娱乐'],
                regionList:['海底捞','希尔顿','洗浴中心'],
})
```
